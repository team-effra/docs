# Granting access in Bupa Providers Online

1. Log into [Bupa Providers Online](https://providersonline.bupa.co.uk/)
2. From the left sidebar menu click **Manage my details** → **Users** → **Add New User**
3. On the top right of the screen click [Find an existing user](https://providersonline.bupa.co.uk/providers-online/account/findUser)
4. Search for `HenryHeath`

<img
  src="/images/bupa-pol-find-user.png"
  alt="Screenshot showing the Find an existing user screen"
  title="Find an existing user"
  style={{ width: "80%", display: "block", margin: "0 auto" }}
/>

5. Click **Add To Provider**
6. If prompted, set the **User Type** to **Delegate**
7. Set the permissions as below:

<img
  src="/images/bupa-pol-permissions.png"
  alt="Screenshot showing the permissions screen"
  title="Permissions"
  style={{ width: "80%", display: "block", margin: "0 auto" }}
/>

8. Click **Save New Permissions**

That's it! You're all set.
